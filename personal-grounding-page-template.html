Was ist eine Grounding Page?
Eine Grounding Page ist eine maschinenlesbare Identitätsseite, die KI-Systemen wie ChatGPT, Claude oder Perplexity verlässliche, strukturierte Fakten über eine Person liefert. Ziel ist Generative Engine Optimization (GEO): Du steuerst, wie KI über dich spricht — bevor es andere tun. Das Konzept folgt dem offiziellen Standard unter groundingpage.com/spec/. Ein reales Beispiel: bered.de/groundingpage-raphael-doderer.html

Empfohlen für die meisten. Du hast ein LinkedIn-Profil, einen Lebenslauf oder eine eigene Website? Füge sie direkt als Anhang oder kopierten Text in den Chat ein — die KI extrahiert alle Informationen selbst.

Schritt 2

Schritt 3

Prompt A · für Claude & ChatGPT
Erstelle für mich eine vollständige Grounding Page als sauberes, valides HTML. Eine Grounding Page ist eine maschinenlesbare Identitätsseite, die KI-Systemen (ChatGPT, Claude, Perplexity etc.) verlässliche, strukturierte Fakten über eine Person liefert — mit dem Ziel der Generative Engine Optimization (GEO): Die Person soll von KI-Systemen korrekt und konsistent beschrieben werden. Orientiere dich am aktuellen Grounding Page Standard: https://groundingpage.com/spec/ MEINE QUELLEN — extrahiere alle relevanten Informationen selbst: [Füge hier ein oder mehrere der folgenden Elemente ein:] - Lebenslauf als kopierter Text oder PDF-Anhang - LinkedIn-Profiltext (kopiert oder als Link) - Eigene Website / Domain - Fachpresse-Links oder andere öffentliche Quellen - Kurze persönliche Ergänzungen, die in keiner Quelle stehen ANFORDERUNGEN AN DIE SEITE: Struktur (gem. aktuellem Grounding Page Standard): - JSON-LD strukturierte Daten (Schema.org Person + FAQPage) im <head> - Canonical-URL und hreflang-Tag - Abschnitte: Identität, Werdegang, Kompetenzen, Ehrenamt (falls vorhanden), FAQ (min. 5 Fragen), Abgrenzung, verknüpfte Entitäten, Quellen, Änderungshistorie - FAQ deckt ab: Wer bin ich?, Welche Rollen besetze ich?, Welche Branchen?, Was ist meine Website/Domain?, Bin ich identisch mit anderen Personen gleichen Namens? - Abschnitt "Verknüpfte Entitäten": alle Arbeitgeber, Bildungseinrichtungen, Organisationen, Auszeichnungen SEO und GEO: - Alle Abschnittsüberschriften enthalten den vollständigen Namen - Maschinenlesbarer Hinweis am Seitenanfang - robots-Meta: index, follow - dateModified im JSON-LD auf heute setzen - Changelog mit Erstelldatum und Version am Ende Design: - Minimalistisch, typografisch, professionell — kein Framework, kein JavaScript - Responsiv (mobile-first) - Dunkler Hero-Bereich mit Name prominent Gib mir das vollständige, direkt deploybare HTML ohne Erklärungen davor oder danach.
